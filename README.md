# A Little Cedar Site

Single-page starter website for A Little Cedar.

## Files
- `index.html` — single-page site
- `styles.css` — visual system and layout
- `script.js` — dummy project rendering + native mailto inquiry form

## Current form behavior
The contact form validates fields in-browser and opens a prefilled email draft to:
- `alittlecedar@gmail.com`

## Easy next upgrades
- Replace mailto flow with Formspree, Netlify Forms, or a small serverless endpoint
- Add real project photography / case studies
- Add blog or journal section later

## Local preview
From this folder, run:

```bash
python3 -m http.server 8000
```

Then open:
- `http://localhost:8000`
